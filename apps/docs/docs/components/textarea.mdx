## Overview

The Textarea component is an enhanced version of the standard HTML textarea with beautiful animations and interactive features. Built with Framer Motion, it provides various animation variants, auto-expanding capabilities, and interactive effects.

## Usage

Import the component:

```tsx
import { AnimatedTextarea } from '@mindfiredigital/ignix-ui';
```

### Basic Usage

```tsx
function BasicTextarea() {
  const [value, setValue] = useState('');
  
  return (
    <AnimatedTextarea
      placeholder="Enter your message"
      variant="clean"
      value={value}
      onChange={setValue}
    />
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | - | Placeholder text for the textarea |
| `variant` | `string` | `'clean'` | Animation variant to apply |
| `value` | `string` | - | The controlled value of the textarea |
| `onChange` | `(value: string) => void` | - | Callback when the value changes |
| `textareaClassName` | `string` | - | Additional CSS classes for the textarea element |
| `labelClassName` | `string` | - | Additional CSS classes for the label element |

### Available Variants

- Basic: `'clean'`, `'glowBorder'`, `'neonGlow'`
- Expansion: `'expandable'`, `'smoothExpand'`
- Features: `'characterCount'`, `'lineHighlight'`, `'syntaxHighlight'`
- Effects: `'particleField'`, `'wave'`, `'spotlight'`, `'liquid'`, `'cosmic'`, `'hologram'`
- Interactions: `'elastic'`, `'rippleEffect'`, `'gradientBorder'`

## Customization

### Custom Styling

You can customize the appearance using CSS classes:

```tsx
<AnimatedTextarea
  placeholder="Custom styled..."
  variant="clean"
  textareaClassName="border-purple-500 focus:border-purple-700"
  labelClassName="text-purple-600"
  value={value}
  onChange={setValue}
/>
```

### Advanced Example with Markdown Preview

```tsx
function MarkdownEditor() {
  const [content, setContent] = useState('');
  const [preview, setPreview] = useState('');
  
  useEffect(() => {
    // Convert markdown to HTML (using your preferred library)
    const html = convertMarkdownToHtml(content);
    setPreview(html);
  }, [content]);

  return (
    <div className="grid grid-cols-2 gap-4">
      <AnimatedTextarea
        placeholder="Write markdown here..."
        variant="syntaxHighlight"
        value={content}
        onChange={setContent}
        textareaClassName="font-mono"
      />
      <div
        className="p-4 border rounded-md"
        dangerouslySetInnerHTML={{ __html: preview }}
      />
    </div>
  );
}
```
