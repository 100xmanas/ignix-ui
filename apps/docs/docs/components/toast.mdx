## Overview

The Toast component is a versatile notification system that provides animated, customizable alerts. Built with Framer Motion, it supports multiple variants, animations, themes, and appearance styles, making it perfect for providing feedback in your applications.

## Usage

First, wrap your application with the ToastProvider:

```tsx
import { ToastProvider } from '@mindfiredigital/ignix-ui';

function App() {
  return (
    <ToastProvider>
      <YourApp />
    </ToastProvider>
  );
}
```

Then use the toast in your components:

```tsx
import { useToast } from '@mindfiredigital/ignix-ui';

function MyComponent() {
  const toast = useToast();
  
  const showNotification = () => {
    toast.addToast({
      message: "Operation successful!",
      variant: "success",
      animation: "slide",
      mode: "light",
      icon: <CheckCircledIcon />
    });
  };
  
  return <button onClick={showNotification}>Show Toast</button>;
}
```

## Props

### ToastProvider Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `maxToasts` | `number` | `100` | Maximum number of toasts to show at once |
| `children` | `ReactNode` | - | Child components |

### Toast Data Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `message` | `string` | - | The message to display in the toast |
| `variant` | `'success' \| 'error' \| 'warning' \| 'info' \| 'default'` | `'default'` | The type of toast |
| `animation` | `'slide' \| 'fade' \| 'bounce' \| 'pop'` | - | Animation style |
| `mode` | `'dark' \| 'light'` | - | Color theme mode |
| `icon` | `ReactNode` | - | Custom icon element |
| `appearance` | `'glow' \| 'gradient'` | `'glow'` | Visual appearance style |
| `gradientColor` | `string` | `'rgba(5, 1, 1, 1)'` | Color for gradient appearance |
| `duration` | `number` | `4000` | Duration in milliseconds before auto-dismiss |
