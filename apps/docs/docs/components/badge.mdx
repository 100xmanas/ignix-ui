## Overview

Badges are small status descriptors that can be used to highlight specific information, such as notification counts, status indicators, or labels. They can include text, numbers, or just a dot indicator.

## Usage

Import the component:

```tsx
import { Badge } from '@mindfiredigital/ignix-ui';
```

### Basic Usage

```tsx
function BasicBadge() {
  return (
    <div className="relative inline-flex items-center">
      <Mail className="h-6 w-6" />
      <Badge text="3" type="primary" />
    </div>
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | - | The text to display inside the badge (required) |
| `type` | `'primary' | 'secondary' | 'success' | 'warning' | 'error'` | `'primary'` | The visual style of the badge |
| `variant` | `'pulse' | 'bounce' | 'tinypop'` | `'tinypop'` | The animation variant of the badge |
| `className` | `string` | - | Additional CSS classes to apply to the badge |
| `children` | `React.ReactNode` | - | The element that the badge will be positioned relative to |

## Customization

### Custom Styling

You can customize the badge appearance by passing custom class names:

```tsx
<Badge 
  text="Custom" 
  className="h-8 w-8 text-sm font-bold"
  type="primary"
/>
```

### With Custom Content

The badge can be used with any React node as children:

```tsx
<div className="relative inline-flex items-center">
  <button className="px-4 py-2 bg-blue-500 text-white rounded">
    Profile
  </button>
  <Badge text="New" type="primary" variant="pulse" />
</div>
```
