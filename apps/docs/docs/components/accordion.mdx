## Usage

Import the components:

```tsx
import * as React from 'react';
import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from '@mindfiredigital/ignix-ui';
```

### Basic Example

```tsx
export function AccordionDemo() {
  return (
    <Accordion type="single" collapsible className="w-full">
      <AccordionItem value="item-1">
        <AccordionTrigger>Is it accessible?</AccordionTrigger>
        <AccordionContent>
          Yes. It adheres to the WAI-ARIA design pattern.
        </AccordionContent>
      </AccordionItem>
    </Accordion>
  )
}
```

## Animation Variants

The AccordionContent component supports different animation variants:

- `fade` (default): Simple fade in/out
- `slideDown`: Slides down from above
- `slideUp`: Slides up from below
- `scaleIn`: Scales in from smaller to larger
- `rotate`: Rotates in 3D space
- `bounce`: Bouncy spring animation
- `flip`: Card flip animation
- `zoomIn`: Zooms in from center
- `elastic`: Elastic spring animation
- `springy`: Springy slide down

Example with custom animation:

```tsx
<AccordionItem value="item-1">
  <AccordionTrigger>Custom Animation</AccordionTrigger>
  <AccordionContent variant="bounce">
    This content will bounce in when expanded.
  </AccordionContent>
</AccordionItem>
```
