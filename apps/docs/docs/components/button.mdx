## Overview

Buttons allow users to take actions with a single tap. They can be customized with different styles, sizes, and animations.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `string` | `default` | Button style variant |
| `size` | `string` | `md` | Button size |
| `animationVariant` | `string` | - | Animation effect |
| `disabled` | `boolean` | `false` | Disable the button |
| `isLoading` | `boolean` | `false` | Show loading state |
| `className` | `string` | - | Additional CSS classes |
| All other button props | - | - | Standard button attributes |

## Examples

### Button with Icon

```tsx
<Button variant="primary">
  <Mail className="mr-2 h-4 w-4" />
  Send Email
</Button>
```

### Loading State

```tsx
<Button isLoading variant="primary">
  Processing...
</Button>
```

### Full Width

```tsx
<Button className="w-full">Full Width Button</Button>
```

### As Link

```tsx
<Button as="a" href="/docs">
  View Documentation
</Button>
```
