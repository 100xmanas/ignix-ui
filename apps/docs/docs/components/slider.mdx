## Overview

The Slider component is a highly customizable input element that allows users to select a value within a range. Built with Radix UI and Framer Motion, it offers various styles, animations, and interactive features.

## Usage

Import the component:

```tsx
import { Slider } from '@mindfiredigital/ignix-ui';
```

### Basic Usage

```tsx
function BasicSlider() {
  const [value, setValue] = useState([50]);
  
  return (
    <Slider
      value={value}
      onValueChange={setValue}
      max={100}
      step={1}
    />
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `SliderVariant` | `'default'` | Visual style variant |
| `animationType` | `SliderAnimationType` | `'none'` | Type of animation effect |
| `animationDuration` | `number` | `0.3` | Duration of the animation in seconds |
| `showValue` | `boolean` | `false` | Whether to display the current value |
| `valuePrefix` | `string` | `''` | Prefix to add before the value |
| `valueSuffix` | `string` | `''` | Suffix to add after the value |
| `defaultValue` | `number[]` | - | Default value of the slider |
| `min` | `number` | `0` | Minimum value |
| `max` | `number` | `100` | Maximum value |
| `step` | `number` | `1` | Step increment value |
| `disabled` | `boolean` | `false` | Whether the slider is disabled |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Orientation of the slider |
| `className` | `string` | - | Additional CSS classes |

### Available Variants

- `default` - Standard slider style
- `minimal` - Simplified, minimal design
- `rounded` - Rounded, softer appearance
- `gradient` - Gradient color effect
- `glass` - Glassmorphism effect
- `outline` - Outlined style
- `shadow` - Shadow effect
- `neon` - Neon glow effect
- `material` - Material Design style
- `neumorphic` - Neumorphic design
- `retro` - Retro-inspired design
- `cyberpunk` - Cyberpunk aesthetic
- `brutalist` - Brutalist design
- `skeuomorphic` - Realistic, skeuomorphic design

### Animation Types

- `none` - No animation
- `slide` - Sliding effect
- `fade` - Fade in/out
- `zoom` - Zoom effect
- `spring` - Spring animation
- `elastic` - Elastic movement
- `parallax` - Parallax effect
- `flip` - Flip animation
- `morph` - Shape morphing
- `hover` - Hover interaction
- `pulse` - Pulsing effect
- `breathe` - Breathing effect
- `wave` - Wave motion
- `rainbow` - Color cycling
- `bounce` - Bouncing motion

## Customization

### Custom Styling

You can customize the appearance using the provided className props:

```tsx
<Slider
  className="w-64"
  trackClassName="h-3 bg-gray-200"
  rangeClassName="bg-blue-500"
  thumbClassName="h-6 w-6 bg-white"
  valueClassName="text-sm font-medium"
  defaultValue={[50]}
  showValue
/>
```

### Custom Range Values

Create a slider with custom range and step values:

```tsx
<Slider
  defaultValue={[0]}
  min={-50}
  max={50}
  step={5}
  showValue
  valuePrefix="Temperature: "
  valueSuffix="Â°C"
/>
```

### With Form Integration

```tsx
import { useForm, Controller } from 'react-hook-form';

function FormSlider() {
  const { control } = useForm({
    defaultValues: {
      volume: [50],
    },
  });

  return (
    <Controller
      name="volume"
      control={control}
      render={({ field }) => (
        <Slider
          value={field.value}
          onValueChange={field.onChange}
          variant="material"
          showValue
          valueSuffix="%"
        />
      )}
    />
  );
}
```
