## Overview

The Spinner component is a versatile loading indicator that comes with multiple variants and customization options. Built with Framer Motion, it provides smooth animations and various styles to indicate loading states in your application.

Import the component:

```tsx

```

### Basic Usage

```tsx

  return <Spinner size={40} color="border-blue-500" />;
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `number` | `40` | Size of the spinner in pixels |
| `color` | `string` | `'border-gray-500'` | Color of the spinner (uses Tailwind CSS classes) |
| `thickness` | `number` | `4` | Thickness of the spinner lines |
| `variant` | `'default' \| 'bars' \| 'dots-bounce'` | `'default'` | Visual style variant |
| `className` | `string` | - | Additional CSS classes |

## Customization

### Custom Styling

You can customize the appearance using Tailwind CSS classes:

```tsx
<Spinner
  className="my-4"
  size={50}
  color="border-indigo-500"
  thickness={6}
/>
```

### With Loading State

Create a loading button with the spinner:

```tsx
function LoadingButton({ isLoading, children }) {
  return (
    <button 
      className="flex items-center gap-2 px-4 py-2 bg-blue-500 text-white rounded"
      disabled={isLoading}
    >
      {isLoading && <Spinner size={20} color="border-white" />}
      {children}
    </button>
  );
}
```

### Loading Card Example

```tsx
function LoadingCard() {
```
