## Overview

The AnimatedInput component provides a rich collection of animated input fields with various effects and styles. Each variant offers unique animations for both the input field and its label, creating engaging user interactions.

## Usage

Import the component:

```tsx
import { AnimatedInput } from '@mindfiredigital/ignix-ui';
```

### Basic Usage

```tsx
function BasicInput() {
  const [value, setValue] = useState('');
  
  return (
    <AnimatedInput
      placeholder="Enter your name"
      variant="clean"
      value={value}
      onChange={(newValue) => setValue(newValue)}
    />
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | - | Placeholder text for the input |
| `variant` | `string` | - | Animation variant to use |
| `value` | `string` | - | Current value of the input |
| `type` | `string` | `'text'` | Type of the input field |
| `onChange` | `(value: string) => void` | - | Called when input value changes |
| `inputClassName` | `string` | - | Additional CSS classes for the input element |
| `labelClassName` | `string` | - | Additional CSS classes for the label element |
| `onFocus` | `() => void` | - | Called when input receives focus |
| `onBlur` | `() => void` | - | Called when input loses focus |

## Customization

### Custom Styling

You can customize the appearance using the provided className props:

```tsx
<AnimatedInput
  placeholder="Custom Styled"
  variant="clean"
  value={value}
  onChange={setValue}
  inputClassName="border-blue-500 focus:ring-2 ring-blue-300"
  labelClassName="text-blue-600"
/>
```

### Combining with Form Libraries

The AnimatedInput component can be easily integrated with form libraries like Formik or React Hook Form:

```tsx
import { useForm } from 'react-hook-form';

function FormInput() {
  const { register, watch } = useForm();
  const value = watch('fieldName');

  return (
    <AnimatedInput
      placeholder="Form Input"
      variant="materialFloat"
      value={value}
      onChange={(newValue) => {
        register('fieldName').onChange({
          target: { value: newValue, name: 'fieldName' },
        });
      }}
    />
  );
}
```
