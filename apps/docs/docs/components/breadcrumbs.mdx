## Overview

Breadcrumbs provide users with a way to navigate back to previous pages or sections in a hierarchical structure. They help users understand their current location within the website and provide quick access to parent pages.

## Usage

Import the component:

```tsx
import { Breadcrumbs } from '@mindfiredigital/ignix-ui';
```

### Basic Usage

```tsx
function BasicBreadcrumb() {
  return (
    <Breadcrumbs
      items={[
        { label: 'Home', href: '/' },
        { label: 'Products', href: '/products' },
        { label: 'Current Page' }
      ]}
    />
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `Array<{ label: string; href?: string }>` | `[]` | Array of breadcrumb items |
| `variant` | `'text' | 'step' | 'custom'` | `'text'` | The visual style of the breadcrumbs |
| `size` | `'sm' | 'md' | 'lg'` | `'md'` | Size of the breadcrumb text |
| `separatorIcon` | `React.ElementType` | - | Custom separator icon component |
| `steps` | `string[]` | `[]` | Array of step labels (used with 'step' variant) |
| `currentStep` | `number` | `0` | Current active step (used with 'step' and 'custom' variants) |
| `shape` | `'rectangle' | 'round' | 'pill' | 'default'` | `'pill'` | Shape of the breadcrumb items |
| `bgColor` | `'blue' | 'gray' | 'green' | 'red'` | - | Background color (used with 'custom' variant) |
| `textColor` | `'white' | 'black' | 'gray' | 'blue'` | - | Text color (used with 'custom' variant) |

## Customization

### Custom Separator

You can customize the separator between breadcrumb items using the `separatorIcon` prop:

```tsx
import { ChevronRight, ArrowRight, Slash } from 'lucide-react';

// Using a different icon
<Breadcrumbs
  items={[...items]}
  separatorIcon={ArrowRight}
/>

// Or a custom element
<Breadcrumbs
  items={[...items]}
  separatorIcon={() => <span className="mx-2 text-gray-400">â€º</span>}
/>
```

### Styling

You can customize the appearance using the provided props or by passing custom class names:

```tsx
<Breadcrumbs
  items={[...items]}
  className="my-4"
  itemClassName="hover:text-blue-700"
  activeItemClassName="font-bold text-blue-800"
/>
```
