## Overview

The card component is a container for text, photos, and actions in the context of a single subject.

## Usage

Import the component:

```tsx
import { Card } from '@mindfiredigital/ignix-ui';
```

### Basic Usage

```tsx
function BasicCard() {
  return (
    <Card>
      This is a basic card.
    </Card>
  );
}
```
